# Rock Paper Scissors Game 🎮

A simple Python mini-game where the user plays Rock, Paper, Scissors against the computer. The computer randomly selects its choice, and the winner is decided based on standard game rules.

## Workflow

1. User enters Rock, Paper, or Scissor
2. Computer selects a random choice
3. Game compares both choices
4. Result is displayed (Win/Lose/Tie)

## Concepts Used

* Python basics
* Conditional statements
* User input
* Random module

This project is ideal for beginners practicing Python through fun interactive games.
